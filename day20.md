# Day 20 – AI Face Puzzle Game

## 🚀 Overview

For Day 20 of the **#60DayClaudeChallenge**, I used Claude to generate a complete browser-based **Face Puzzle Game** as a single HTML application.

The application captures a user's photo through the webcam, converts it into a sliding puzzle, and allows the user to solve it using drag-and-drop or touch interactions.

---

# ✨ Features

* 📷 Webcam integration for live photo capture
* 🧩 Dynamic puzzle generation from captured image
* 🎮 Multiple difficulty levels

  * 3×3
  * 4×4
  * 5×5
* 🖱️ Drag-and-drop support
* 📱 Touch interaction support
* ⏱️ Live timer
* 🔢 Move counter
* 🏆 Local leaderboard
* 💻 Responsive UI
* 🎨 Improved visual design and smooth animations

---

# ▶️ How to Run

1. Download or clone the repository.
2. Open `face-puzzle-game.html` in a modern web browser.
3. Allow webcam permissions when prompted.
4. Capture your photo.
5. Select a puzzle difficulty.
6. Start solving the puzzle.
7. Complete the puzzle to view your time, moves, and leaderboard ranking.
