# Day 17 – Vehicle Cost Analysis Dashboard using Claude

## Objective

For Day 17 of the **#60DayClaudeChallenge**, I used Claude to generate an interactive HTML dashboard for vehicle cost analysis from a CSV dataset. The dashboard visualizes ownership costs, fuel expenses, maintenance trends, environmental impact, and E85 fuel economics.

---

## Task Completed

* Downloaded the provided vehicle dataset (CSV)
* Gathered vehicle information
* Updated the prompt with vehicle details
* Uploaded the dataset into Claude
* Generated a complete HTML dashboard
* Reviewed KPI cards and interactive visualizations
* Analyzed fuel comparisons and maintenance costs
* Explored E85 economics and break-even calculations
* Saved the generated HTML dashboard
* Opened the dashboard in a web browser
* Captured dashboard screenshots
* Uploaded all deliverables to GitHub

---

## Dashboard Features

* Interactive KPI Cards
* Fuel Cost Comparison
* Maintenance Cost Analysis
* Environmental Impact Metrics
* E85 Economics Analysis
* Break-even Calculator
* SVG Charts and Gauges
* Fuel Consumption Visualization
* Ownership Cost Breakdown

---

## Key Insights from the Dataset

One of the most interesting findings was that **fuel price alone does not determine the overall operating cost**. While some fuel options appeared cheaper initially, the dashboard demonstrated that maintenance costs, fuel efficiency, and annual driving distance significantly affected total ownership expenses.

The E85 break-even analysis also highlighted that a lower fuel price does not always translate into lower long-term costs, emphasizing the importance of considering multiple variables before making fuel-related decisions.

---

## Key Learnings

* Data visualization makes complex datasets much easier to interpret.
* Interactive dashboards reveal trends that are difficult to identify from raw CSV files.
* KPI cards provide an effective summary of important metrics.
* Visual comparisons simplify cost-benefit analysis.
* Claude can rapidly generate professional HTML dashboards from structured datasets.
