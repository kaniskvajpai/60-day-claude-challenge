# Day 36 – Cognitive Pattern Explorer

## Objective

Built an interactive **Cognitive Pattern Explorer** application using Claude AI to help users understand how their thinking patterns change under different mental states. The application explores decision-making, cognitive biases, emotional responses, and reflection through an engaging multi-chapter experience.

---

# Project Overview

The application guides users through several interactive chapters where they respond to everyday situations, complete drag-and-drop exercises, organize thinking sequences, and receive a personalized Reflection Journal based on their choices.

Users can experience the application in both **Calm Mode** and **Stress Mode** to compare how different mental states influence thinking.

---

# Features

* Calm Mode and Stress Mode
* Interactive scenario-based decisions
* Drag-and-drop cognitive activities
* Thinking sequence arrangement
* Personalized Reflection Journal
* Thinking tendency analysis
* Progress tracking
* Replay with alternate cognitive mode
* Responsive UI
* Fully client-side HTML application

---

# Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Claude AI

---

# Application Workflow

1. Select Calm Mode or Stress Mode
2. Complete Chapter 1 everyday scenarios
3. Finish Chapter 2 drag-and-drop exercises
4. Arrange cognitive sequences in Chapter 3
5. Generate Reflection Journal
6. Review Thinking Profile
7. Replay using the opposite mode
8. Compare behavioral differences
