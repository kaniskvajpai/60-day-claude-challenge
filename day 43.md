## Key Learnings

- AI workflows become more effective when broken into modular stages.
- Proper prompt engineering significantly improves output quality.
- Automation opportunities should be identified during the design phase rather than after implementation.
- Combining multiple AI tools can create efficient end-to-end workflows.
- Documentation and workflow visualization improve collaboration.

## Reflection

This project demonstrated how AI can be systematically integrated into real-world workflows. Instead of viewing AI as a single tool, it should be treated as a complete workflow partner that supports planning, execution, optimization, and continuous improvement.
